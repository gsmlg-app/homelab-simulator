# AI Asset Generation Prompts

## Global Prompt Rules
- Style: homelab sci-fi, top-down 3/4 perspective.
- Format: PNG with transparent background.
- Lighting: cool cyan/teal accents, soft ambient shadows.
- Avoid: text, watermarks, logos, UI text.
- Output size: 32x32 for tiles; props at listed tile multiples.

## Tiles
### Floor Metal
Prompt:
```
Top-down 3/4 pixel art floor tile, clean brushed metal panels with subtle bolts, cool cyan accent lights, seamless 32x32 tile, transparent background.
```
